### Local Installation

1. Clone this repository:
```bash
git clone https://github.com/YOUR-USERNAME/mortgage-calculator.git
cd mortgage-calculator
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
streamlit run mortgage_app.py
```

4. Open your browser to `http://localhost:8501`

## Input Parameters

- **Loan Principal Amount**: Total loan amount ($1,000 - $10,000,000)
- **Annual Interest Rate**: Interest rate as percentage (0% - 20%)
- **Loan Term**: Length of loan in years (1 - 50 years)

## Outputs

- Monthly payment amount
- Total amount paid over loan term
- Total interest paid
- Principal vs Interest visualization over time
- Remaining balance chart
- Detailed amortization schedule with:
  - Payment number
  - Payment amount
  - Principal payment
  - Interest payment
  - Remaining balance
